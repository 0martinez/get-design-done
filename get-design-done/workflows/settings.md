# Workflow: settings

Configure GDD workflow preferences.

---

## Process

1. Load current config.json
2. Display current settings
3. Allow changes to:

### Adapter
- paper / pencil / generic
- Re-detect available tools

### Workflow Mode
- interactive (confirm each step)
- express (auto-approve after initial questions)

### Agent Toggles
- Researcher: on/off
- Moodboard: on/off
- Critic: on/off
- Accessibility checker: on/off

### Preferences
- Color scheme: light / dark
- Commit docs: true/false (whether to git commit .design/ files)

4. Write updated config.json
5. Show summary of changes
